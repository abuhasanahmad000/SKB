**Fokus utama:** Validasi ide, pembuktian potensi pasar, daya tarik keunikan produk.

- **EMPHASIS** di sub-tab "Anggaran Modal Awal" → renovasi, perizinan, stok awal, promosi pembukaan (start-up requirements masif)
